# How to run and compile game

**Step 1. Installation and Downloading** 
- Make sure raylib is installed
- Download the Main_Game file 
- Download all the textures in *resources* 

**Step 2. Saving the files properly** 
- Save your file in Visual Studio code in C and give it a name, ending in *.c*
- An example for the file name of the game is *main.c*
- Make your folder where you save your file has the resources folder with all textures inside (again in Visual Studio code)

**Step 3. Running and Compiling**
- Compile and run the code by pressing *Fn* and *F5* together

This should work for both Windows and MacOS
